CRC Phage Project: F. nucleatum-Driven Autocrine and Immune Tolerance Axis

Overview
This repository contains the computational pipeline investigating how the oncomicrobe Fusobacterium nucleatum rewires the transcriptomic landscape of Colorectal Cancer (CRC). Using unbiased systems biology approaches on publicly available bulk RNA-seq data (GSE90944), this project maps the non-canonical hyperactivation of host master regulators and their downstream effector ligands. 

Biological Hypothesis
I hypothesize that F. nucleatum utilizes a dual-pathway survival mechanism to manipulate the Tumor Microenvironment (TME):
1. Autocrine Growth: TCF4-driven upregulation of HBEGF to stimulate an HBEGF-EGFR autocrine growth loop.
2. Immune Tolerance: Synchronized Wnt/Hedgehog-driven secretion of GDF15 to engage GFRAL/RET receptors on tumor-associated macrophages (TAMs), paralyzing local immune surveillance.

Computational Workflow
The analysis pipeline moves from raw transcriptomic counts to high-level network inference:
- Data Acquisition: Processing of binary F. nucleatum-infected vs. uninfected HT-29 CRC transcriptomes (GSE90944).
- Differential Expression: Identification of pathogen-induced host gene signatures using DESeq2.
- Transcription Factor Inference: Global master-regulator analysis utilizing decoupleR to computationally isolate Wnt/beta-catenin (TCF4) and Hedgehog (GLI1/GLI2) hyperactivation.
- Network Mapping: Conceptual ligand-receptor communication modeling utilizing igraph to visualize the downstream HBEGF and GDF15 structural networks.

Key Findings
- Figure 10: Unbiased TF activity scores confirming severe TCF4 and GLI1 perturbation post-infection.
- Network Schematic: A conceptual TME communication map formally integrating the bulk transcriptomic discoveries into a spatiotemporal hypothesis for tumor escape.

Repository Structure
- scripts/: Contains all modular R scripts for the analysis pipeline (e.g., 17_scRNAseq_Conceptual_Network.R).
- figures/: High-resolution, publication-ready outputs generated by the scripts.
- data/: Processed count matrices and metadata utilized in the analysis.

Future Directions
Because current global transcriptomic repositories (such as GSE90944) are restricted to single-timepoint, binary-dose snapshots, the computational hypotheses generated in this repository serve as the foundation for targeted in vitro spatiotemporal validation. Proposed next steps include dense temporal transcriptomics (RT-qPCR) and translational validation across an F. nucleatum MOI gradient.
